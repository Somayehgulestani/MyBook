MyBooks App
A clean and interactive React app for managing and exploring books. Users can search books by title and see statistics for the books they have read.
#Features
Search Books: Filter the list by book title 
Books List: Display available books with cover, title, author, and year
Read Books: Show books that have been read, including user rating and average book rating
Statistics: Calculate and display average user rating and average book rating
# Built With
React
JavaScript
Tailwind CSS
#Purpose
Designed to practice:
React state management
List filtering and dynamic rendering
Handling component props
Calculating and displaying averages
